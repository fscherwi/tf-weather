# Usage
## Simple show current weather

```sh
$ weather
```

## Configure

```sh
$ weather get
```

### Options
#### Custom HOST

```sh
$ weather get -h <your-host>
```

#### Custom PORT

```sh
$ weather get -p <your-port>
```

## Show config

```sh
$ weather info
```

## Advanced functions
### Show live weather

```sh
$ weather live
```
