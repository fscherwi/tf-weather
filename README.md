## Shows the current weather from Tinkerforge Sensors!
# Install
You may require sudo!

```sh
 $ npm install -g tf-weather
```

# Usage
## Simple show current weather

```sh
 $ weather
```

## Configure

```sh
 $ weather get
```

## Show config

```sh
 $ weather info
```

## Advanced functions
### Show live weather

```sh
 $ weather live
```

### Custom HOST

```sh
 $ weather -h <your-host>
```

### Custom PORT

```sh
 $ weather -p <your-port>
```
